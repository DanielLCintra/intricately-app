# Intricately App - By Daniel Cintra

### To run project in development mode:

```sh
npm run start
```

or

```sh
yarn start
```

### To build project

```sh
npm run predev
```

or

```sh
yarn predev
```

### To run tests

```sh
npm run test
```

or

```sh
yarn test
```

### Utilized Libraries

To develop the project were used:

- Vue.js
- Vue Router
- Vuex
- Sass
